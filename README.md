# PVAMU NSF Workshop 2026: Applied Policy Research and Data Analysis

Welcome to the workshop! This repository contains the materials, scripts, and documentation for our session on applied econometrics and state-level policy analysis. 

This guide is designed to help you navigate the folders and set up your environment to reproduce the analysis presented in the slides.

## 🚀 Getting Started

### 1. Presentation Slides
The core concepts, research designs, and findings are covered in the main presentation file:
- **File:** `slides/ebooth_slides_20260519.pdf`
- **What's inside:** An overview of causal inference methods (Difference-in-Differences, Panel Data), data sources, and policy applications in the Texas context.

### 2. Data Requirements (Important!)
Due to licensing and size constraints, the raw datasets are **not** included in this repository. You will need to download the data from the following sources as mentioned in the workshop notes:
- **ICPSR (Inter-university Consortium for Political and Social Research):** Used for the PRAMS (Pregnancy Risk Assessment and Monitoring System) analysis. Look for the dataset ID mentioned in the slides for Examples 1 and 2. 
    - *** Medicaid expansion data: https://www.openicpsr.org/openicpsr/project/114516/version/V1/view?path=/openicpsr/114516/fcr:versions/V1 
    - *** PRAMS data: https://www.openicpsr.org/openicpsr/project/221043/version/V1/view;jsessionid=0851564D4EC837B60874212A234350E7
- **Zelma.ai:** Used for the education performance data and NAEP/STAAR comparisons (Example 3)  https://www.zelma.ai/data

Please ensure you save these datasets in the `scripts/` directory or update the file paths in the Stata do-files accordingly.

## 📂 Repository Structure

- `slides/`: Contains the PDF presentation and supporting images.
- `scripts/`: This is where the magic happens. You'll find three primary Stata do-files:
    1. `workshop_walkthrough.do`: A foundational script to get you started with the workflow.
    2. `221043-V2/prams_2022_analysis.do`: Detailed analysis of maternal health indicators using PRAMS data.
    3. `edc-3.0-texas/analyze_performance.do`: Comparative analysis of Texas education performance data.
- `figures/`: Output directory for charts and visualizations generated by the scripts.
- `tables/`: Econometric results and descriptive statistics in `.tex` format.

## 📊 Visualizing Results: Sparkta Dashboard
In the `scripts/` folder, you will find `medicaidsummary.html`. This is a **Sparkta dashboard** generated by our analysis. 
- To view it, simply open the file in any web browser. 
- It provides an interactive summary of the Medicaid-related findings discussed in the workshop.

## 🛠️ How to Run the Analysis
1. **Open Stata:** Ensure you have Stata installed (version 16+ recommended).
2. **Set Working Directory:** Change your working directory in Stata to the root of this repository.
3. **Run Scripts:** Open and execute the do-files in the `scripts/` directory. Each script is commented to explain the econometric logic and data cleaning steps.

## ❓ Need Help?
If you run into issues with data paths or Stata commands, refer back to the slides or reach out during the Q&A session. Happy coding!
